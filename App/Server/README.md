# GalaxyAndExoplanet
## **1. Create a virtual python environment**

```bash 
python -m venv ./pathtofolder
```

## **2. Install the necessary pip packages using the command**

```bash
pip install -r requirements.txt
```

## 3. **Run the app.py file**

```bash
python app.py
```

## **4. Start the flask server**

```bash
flask run
```
